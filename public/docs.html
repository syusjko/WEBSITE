JavaScript SDK

Prospector JavaScript SDK allows you to integrate AI-powered ads into your web application.

ℹ
Before getting started, you'll need to create a publisher account in our dashboard and obtain a publisher ID.

Installation

Copy the snippet below and replace <YOUR_PUBLISHER_ID> within the <head> tags of your HTML file. You can find your publisher ID in your dashboard settings.

<script async src="https://app.prospector.ai/js?token=<YOUR_PUBLISHER_ID>"></script>

Injecting ads

There are two steps involved in serving Prospector ads: sending chat interaction data and specifying ad placement locations. Our system analyzes user-AI conversations to deliver contextually relevant advertisements.

1. Send chat interaction data

You can use the window.prospector.process method to send chat interaction data to our system.

window.prospector.process({ message: "User's message content", context: "Additional context information" });
